//TaskManager API.

TaskManager es una API RESTful desarrollada en ASP.NET Core para gestionar información relacionada con la gestion de tareas. 
La arquitectura está basada en buenas prácticas como separación por capas (Aplicación, Dominio, Infraestructura), 
uso de Entity Framework Core y patrones genéricos como repositorios y servicios.


// Propósito.

El objetivo principal de esta API es ofrecer una solución modular, escalable y mantenible para manejar recursos, 
incluyendo la posibilidad de crear tareas dinámicas sobre entidades mediante el uso de clases genéricas.

// Configuración del proyecto.

Requisitos.

- .NET 8 SDK
- SQL Server
- Visual Studio 2022

//Pruebas.

Para probar el API luego de configurado el proyecto correr el mismo y pasar los parametros corresponidentes en cada endpoint y validar las respuestas.
